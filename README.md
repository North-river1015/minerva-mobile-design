# [Minerva](https://minerva-project.org/)
> **日本の政治を変える**

Minervaは、国政政治家の公約内容と、実行の有無を確認できるサイトです。　　
「ミネルヴァのフクロウは黄昏に飛び立つ」との言葉から名前を取りました。

### 1. データの追加・修正（非エンジニアの方も歓迎！）
Markdownです。
選挙公報から公約を載せる際は  
> content/prefectures/都道府県/選挙区の番号/    

にてお願いします。  

選挙区.mdがテンプレートになっています。公約か否かの判断は**rubric.md**を元に。**rubric**をより改善する手を見つけた方は遠慮せずにPull requestしてください。      


政治家の選挙公報を追加する際は
>content/shu/都道府県/選挙区の番号/政治家の名前.md

にてお願いします。政治家.mdがテンプレートです。テンプレート改善も歓迎します！

## 理念

Minervaは、日本の国政政治家が掲げた公約の実行状況を、透明性の高いプロセスで客観的に評価するプラットフォームを目指しています。

### 客観性と中立性
>特定の政治団体への支持・不支持を目的としません。個人の主観的な努力や意図ではなく、公開されている情報に基づき何が成し遂げられたかという**客観的に検証可能な結果**のみに焦点を当てます。

# なぜこのサイトが必要なのか。

選挙公報には「検討」「努力する」といった曖昧な言葉が並び、有権者が政治家の真意や実績を判断するのは困難です。有権者が政治家を判断するための材料は、驚くほど不足しています。
現状の選挙では、具体的な政策よりも名前の連呼や雰囲気、あるいはポピュリズム的な訴えが優先され、投票後の検証はほとんど行われません。
それはある種、政治家の「有権者は政策を理解しないだろう」という、国民の知性に対する負の信頼に基づいた振る舞いにも見えます。

政治家が「国民に伝わらない」と諦め、有権者が「どうせ変わらない」と棄権する。この無責任なポピュリズムとおまかせ民主主義のループを、データと技術の力で打破したいと考えています。

誠実に公約を掲げ、実行する政治家が正当に評価される土壌を作ること。 実現できなかった際には、政治家がその理由を自らの言葉で説明する責任を求めること。

「おまかせ民主主義」から、事実に基づき自ら考える民主主義へ。 まだ小さなベータ版ですが、この構想を全国に広げるために、あなたの力を貸してください。
